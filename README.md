# Learn PureScript (0.12)

Basic starter kit for learning PureScript.

## Quickstart

### Install

```
npm install
```

### Run

```
# run all exercises
npm start

# run specific exercise
npm start <exercise>
```

### Add new dependency

To add a new dependency, add its name to `psc-package.json` `depends` list
and run `npm install`.
